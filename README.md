# harshilsuthar.com

Terminal-themed personal portfolio. Single self-contained `index.html` (inline CSS + vanilla JS, no build step).

- **Live:** https://harshilsuthar.com
- **Hosting:** Cloudflare Pages (Git-connected — pushes to `main` auto-deploy)
- **Contact form:** Web3Forms (submissions go to a private inbox; email is not exposed on the page)

## Deploy

Cloudflare Pages builds straight from this repo:

| Setting | Value |
|---|---|
| Framework preset | None |
| Build command | *(empty)* |
| Output directory | `/` |

Edit `index.html`, commit, `git push` → live in ~30s.
